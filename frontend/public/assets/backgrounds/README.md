# Background Assets

Large PNG images for full-screen or scene backgrounds.

Needed:

- `boot-world.png` - connection and country-selection screen background. Wide fantasy world map, 16:9 or wider, recommended 2560x1440 or 3200x1800.
- `main-map-backdrop.png` - global map screen background if we use a painted bitmap behind interactive overlays, recommended 2560x1440.
- `economy-map-panel.png` - smaller kingdom/economy map shown inside the economy dashboard, recommended 1200x760.
- `castle-yard-backdrop.png` - optional 2D backdrop behind the Babylon.js castle scene, forest/walls atmosphere, recommended 2560x1440.
- `panel-noise.png` - subtle dark parchment/metal noise texture for UI panels, seamless if possible, recommended 512x512 or 1024x1024.
- `gold-edge-texture.png` - optional trim texture for gold borders and ornate separators, recommended 512x128.

If you only provide one painted world map, name it `boot-world.png`; I can reuse
and crop it for boot, map, and minimap screens.
