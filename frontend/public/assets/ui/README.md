# UI Assets

Reusable UI PNG assets for the medieval/dark strategy interface.

Most panels, borders, and buttons can be done with CSS. PNGs are useful for
ornate details, icons, and hand-painted accents.

box-shadows, gradients, and subtle noise textures.
