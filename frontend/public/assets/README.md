# MF.Game Asset Checklist

This folder is reserved for PNG assets used by the frontend prototype.

Use lowercase kebab-case names, for example `riverland-crest.png`.
Prefer transparent PNG for icons, crests, portraits cutouts, and building previews.
Prefer 2x resolution when possible so UI can stay crisp on high-DPI screens.

Suggested folders:

- `backgrounds/` - large screen and scene backgrounds.
- `branding/` - logo, crown mark, decorative dividers.
- `buildings/` - building list icons and 2D previews.
- `events/` - event log and notification icons.
- `kingdoms/` - faction crests, ruler portraits, and country cards.
- `map/` - painted map images, overlays, markers, minimap assets.
- `resources/` - resource, economy, and production icons.
- `ui/` - reusable interface icons, trims, buttons, frames, panels.

3D models are not required for the current prototype. Castle buildings can be
assembled from Babylon.js primitives, while this folder provides 2D UI assets
for menus, cards, previews, icons, and backgrounds.
