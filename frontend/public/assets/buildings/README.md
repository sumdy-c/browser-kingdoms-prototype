# Building Assets

PNG assets for building menus, cards, and right-side previews. 3D models are not
needed; the castle scene can use styled Babylon.js cubes/cylinders.

Needed building icons, transparent PNG, recommended 128x128:

- `house-icon.png`
- `farm-icon.png`
- `lumberyard-icon.png`
- `quarry-icon.png`
- `barracks-icon.png`
- `archery-range-icon.png`
- `warehouse-icon.png`
- `watchtower-icon.png`
- `market-icon.png`

Needed building previews, transparent PNG or small rendered cards, recommended
512x384:

- `house-preview.png`
- `farm-preview.png`
- `lumberyard-preview.png`
- `quarry-preview.png`
- `barracks-preview.png`
- `archery-range-preview.png`
- `warehouse-preview.png`
- `watchtower-preview.png`
- `market-preview.png`

Optional build-mode overlays:

- `grid-cell-valid.png` - green/valid placement cell.
- `grid-cell-invalid.png` - red/invalid placement cell.
- `grid-cell-hover.png` - neutral hover cell.
- `castle-banner-blue.png` - banner texture/PNG for walls and UI.
- `castle-banner-red.png`
- `castle-banner-green.png`
- `castle-banner-gold.png`

If previews are not available, I can generate the in-scene buildings with
Babylon primitives and use only the icons in the UI.
