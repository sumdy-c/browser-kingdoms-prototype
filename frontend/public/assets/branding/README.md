# Branding Assets

Logo and decorative brand assets.

Needed:

- `logo-full.png` - MF.Game wordmark with crown, transparent background, recommended 700x180.
- `logo-crown.png` - standalone crown mark, transparent background, recommended 256x256.
- `logo-small.png` - compact logo for tight headers, transparent background, recommended 360x96.
- `divider-gold.png` - thin decorative divider used around headings, recommended 512x48.
- `button-flourish-left.png` - optional left ornament for large CTA buttons, transparent background.
- `button-flourish-right.png` - optional right ornament for large CTA buttons, transparent background.

The text logo can also be recreated with CSS if you do not have a final logo PNG,
but the standalone crown is useful for buttons, loading states, and badges.
