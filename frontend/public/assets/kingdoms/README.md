# Kingdom Assets

Faction assets for country selection, side panels, map shields, and ruler cards.

For each kingdom, provide:

- Crest/shield PNG, transparent background, recommended 256x320.
- Ruler portrait PNG/JPG, recommended 512x640 or 768x960.
- Country card image PNG/JPG, recommended 512x360.
- Optional small banner PNG, transparent background, recommended 512x160.
