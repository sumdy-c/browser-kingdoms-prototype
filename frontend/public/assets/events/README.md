# Event Assets

Transparent PNG icons for event log, active events, notifications, and queue
items. Recommended size: 96x96 or 128x128.

Needed:

- `event-trade.png` - caravan/trade income.
- `event-harvest.png` - harvest/festival/food bonus.
- `event-bandits.png` - border danger or bandits.
- `event-research.png` - completed research.
- `event-diplomacy.png` - alliance or treaty.
- `event-army.png` - invasion or battle.
- `event-construction.png` - building completed.
- `event-warning.png` - generic warning.
- `event-success.png` - generic positive result.
- `event-scroll.png` - generic world news.

Optional notification badges:

- `badge-red.png`
- `badge-gold.png`
- `badge-green.png`
